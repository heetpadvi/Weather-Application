About

WeatherNow uses WeatherApi's weather API to get the current weather reports of desired location. WeatherAPI.com provides access to free weather and geo data via a JSON/XML restful API. It allows developers to create desktop, web and mobile applications using this data very easy.
Realtime, Hourly, Daily and 15 min interval weather forecast, Historical Weather, Air Quality Data, Bulk Request, Astronomy, Search, Sports and much more
